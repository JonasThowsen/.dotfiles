# Phoenix LiveView documentation references

Consult these sources when working on LiveView tasks.

## Official docs

Start with current package docs on HexDocs:
- https://hexdocs.pm/phoenix_live_view/
- https://hexdocs.pm/phoenix/

Use these for:
- LiveView lifecycle
- assigns and state
- events
- forms
- streams
- uploads
- components
- navigation and patching
- templates and HEEx behavior

## Phoenix generators and guides

If relevant, also check Phoenix guides and current docs for the version used by the project.

## Local codebase check

After consulting official docs, inspect the local repository for:
- existing LiveViews
- function components and live components
- form patterns
- event naming conventions
- navigation patterns
- JS hook usage, if any

Official docs provide current APIs and idioms; the local codebase provides project consistency.
