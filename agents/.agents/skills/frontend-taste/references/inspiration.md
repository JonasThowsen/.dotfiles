# Local frontend inspiration

Read this file when you need visual direction, layout ideas, or examples of polished CSS composition.

## Inspiration source

Primary local inspiration path:

- `~/coding/frontend-inspiration`

Inspect that directory for examples of strong layout, spacing, typography, and component composition.

## How to use the inspiration

Treat the contents as design inspiration, not as templates to copy blindly.

Look for:
- spacing rhythm
- layout structure
- responsive composition
- restrained color usage
- effective section separation without excessive borders
- situations where cards are used sparingly and appropriately

## Tailwind-specific note

Many inspiration examples may come from Tailwind-based work.

If the current project already uses Tailwind CSS:
- it is fine to borrow implementation ideas more directly

If the current project does not use Tailwind CSS:
- translate the ideas into plain CSS
- preserve the design intent rather than the Tailwind utility structure
- do not introduce Tailwind just because the inspiration uses it

## Taste filter

Prefer inspiration that feels:
- spacious
- calm
- deliberate
- mobile-friendly
- structurally strong before decorative details are added

Be skeptical of inspiration that depends mostly on:
- heavy gradients
- excessive cards
- too many borders
- visually loud effects masking weak layout
