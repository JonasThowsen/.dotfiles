# Elixir documentation references

Consult these sources when working on Elixir tasks.

## Official language and standard library docs

Start with:
- https://elixir-lang.org/docs.html
- https://hexdocs.pm/elixir/

Use these for:
- language behavior
- standard library modules
- syntax and semantics
- official guides and current examples

## Package documentation

For libraries, prefer the package's current HexDocs pages.

General entry point:
- https://hexdocs.pm/

If the project depends on a package, look up that package's current HexDocs before relying on memory.

## Local codebase check

After consulting the docs, inspect the local repository for:
- existing module shapes
- naming conventions
- tuple/error return patterns
- test style
- supervision/context boundaries if relevant

Official docs provide correctness; the local codebase provides fit.
